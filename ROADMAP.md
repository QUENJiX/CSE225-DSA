# 3-Week Semester Break Roadmap

This is a practical bridge from "I know basic C++" to "I can survive and enjoy CSE225".

## Week 1: C++ Refresh + OOP

- Day 1: input/output, types, strings, vectors.
- Day 2: functions, references, const correctness.
- Day 3: pointers, dynamic memory, stack vs heap.
- Day 4: classes, objects, constructors, destructors.
- Day 5: inheritance, polymorphism, virtual functions.
- Day 6: templates, operator overloading, exceptions.
- Day 7: ADTs, interface vs implementation, domain-analysis guide, then solve 6 to 8 easy problems.

## Week 2: Core DSA Patterns

- Day 8: Big-O, operation tradeoffs, and binary search.
- Day 9: linked lists, stacks, queues, deques.
- Day 10: recursion basics.
- Day 11: backtracking with subsets and permutations.
- Day 12: sorting algorithms and STL sorting.
- Day 13: binary trees and BSTs.
- Day 14: solve 8 to 12 mixed easy/medium problems.

## Week 3: Graphs + DP + Grokking Algorithms Finish

- Day 15: BFS, DFS, connected components.
- Day 16: shortest paths and topological sort.
- Day 17: heaps, priority queues, union-find.
- Day 18: dynamic programming: memoization.
- Day 19: dynamic programming: tabulation.
- Day 20: mock contest, 3 to 5 problems.
- Day 21: review notebook, re-solve missed problems, clean templates.

## Weekly Review Questions

- Which ADT or data structure matched the problem domain best?
- What pattern did I see more than once?
- Which bug cost the most time?
- Which STL tool saved the most code?
- Which problem should I re-solve in 7 days?
