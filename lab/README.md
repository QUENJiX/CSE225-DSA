# CSE225L Lab Workspace

Use this folder for your own lab work. Keep official lab manuals and old submissions in `../Materials/CSE225L/`; keep your fresh code and notes here.

## Folders

- `00_lab_notes/` - notes from each lab manual before coding.
- `01_lab_practice/` - rough implementations while learning the lab topic.
- `02_lab_submissions/` - clean final versions for submission.
- `03_assessment_rebuilds/` - re-coded versions of previous assessments without looking at the answer first.

## Lab Habit

1. Read the lab PDF.
2. Write the ADT/interface in plain English.
3. Implement the smallest working version.
4. Test boundary cases.
5. Move the clean version into `02_lab_submissions/`.
