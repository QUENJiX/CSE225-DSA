# Problem Solving Pattern Notes

## Two Pointers

Use when the array/string is sorted or when a window can expand/shrink.

## Sliding Window

Use for contiguous subarray/substring problems with a running condition.

## Binary Search On Answer

Use when the answer is numeric and the feasibility check is monotonic.

## BFS

Use for shortest path in an unweighted graph/grid.

## DFS

Use for traversal, components, recursion trees, and backtracking.

## Dynamic Programming

Use when:

- The problem has overlapping subproblems.
- A state can summarize all past decisions needed for the future.
